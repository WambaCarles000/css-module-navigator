# Change Log

All notable changes to the "css-module-navigator" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.1] - 2026-01-03

### Added
- Quick navigation to CSS Module files from imports
- Support for relative imports (`./`, `../`)
- Support for path aliases via `jsconfig.json` or `tsconfig.json`
- Support for multiple extensions: `.module.css`, `.module.scss`, `.module.sass`
- Keyboard shortcut `Ctrl+Alt+C` (or `Cmd+Alt+C` on Mac)
- Context menu and command palette integration
- Automatic search for configuration files in subdirectories
- Support for trailing commas in JSON configuration files
